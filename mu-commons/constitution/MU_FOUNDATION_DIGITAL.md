# MU Foundation 全デジタル設立手順 — No Hanko, No Paper

**Status**: 設立計画書 (2026-05-18 起草)
**目的**: 一般財団法人 MU Foundation を **マイナンバーカード電子署名** だけで設立し、登記までハンコゼロ・紙ゼロで完結する。
**根拠**: §29 The Discipline of Silence — MU の運営思想と整合。アナログな儀式を排除。

---

## 結論

**2026 年現在、一般財団法人の設立は全工程デジタルで可能**。ただし以下 4 ステップそれぞれに固有の電子化要件がある。

| ステップ | 電子化手段 | 必要なもの | コスト |
|---|---|---|---:|
| ① 定款作成 | テキストエディタ → PDF/A 化 | なし | ¥0 |
| ② 電子定款認証 | 公証役場の **遠隔 web 会議認証** + **マイナンバーカード電子署名** | マイナンバーカード, IC カードリーダ (or スマホ NFC), Adobe Acrobat 等の **電子署名対応 PDF ソフト** | ¥52,000 (印紙税ゼロ、紙定款の場合は ¥40,000 印紙が必要) |
| ③ 基本金拠出 | 銀行 web 振込 | 基本金 ¥3M、 設立時取締役の個人口座 | ¥3,000,000 |
| ④ 登記申請 | **登記ねっと** (法務省オンライン) + マイナンバーカード電子署名 | 同上 + 登録免許税電子納付 | ¥60,000 (登録免許税) |
| **合計** | | | **約 ¥3,112,000** |

**ハンコ使用回数: 0** ・ **紙提出回数: 0** ・ **窓口訪問: 0**

## ステップ詳細

### ① 定款作成 (Articles of Incorporation)

設立者 (1 名以上、 一般財団法人は **3,000 万円以上の拠出が必要 → 改正により ¥300 万に緩和済み**、 基本金 ¥3M で要件達成) が以下を含む定款を作成:

- 名称: **一般財団法人 MU Foundation**
- 主たる事務所: 北海道川上郡弟子屈町 (企業版ふるさと納税認定自治体)
- 目的: ① 地方創生 ② 教育機会の創出 ③ アーティスト・起業家支援 ④ 武道・身体文化の継承
- 拠出財産: 基本金 ¥3,000,000
- 評議員・理事・監事の選任方法
- 公告の方法 (電子公告で OK)
- 事業年度

**PDF/A-1 形式** で保存 (e-Tax / 登記ねっと の要件)。Microsoft Word → 名前を付けて保存 → PDF/A、 または Adobe Acrobat で変換。

### ② 電子定款認証 (公証役場、 遠隔 Web 会議方式)

2020 年から **公証役場の遠隔 web 会議方式** が認められ、 全工程オンライン完結可能:

1. 最寄りの **電子公証取扱の公証役場** に事前に電話 (弟子屈町なら釧路法務局管内の **釧路公証役場** が候補)
2. 「遠隔 web 会議による電子定款認証を希望」と伝える
3. 定款 PDF にマイナンバーカードで電子署名 → 公証役場にメール送付
4. 指定された日時に **Microsoft Teams** で web 会議 (本人確認 + 認証手続)
5. 認証済み電子定款を返送 (メール添付)
6. **認証料 ¥52,000** をペイジー/ネット銀行振込

**紙定款だと印紙税 ¥40,000 が追加** → 電子定款選択で実質 ¥40,000 節約。

### ③ 基本金拠出 ¥3,000,000

設立時理事の代表者個人口座へ web 振込で集約。

- 銀行: 任意 (Enabler Inc. のメインバンクが楽)
- 振込人名義に "MU Foundation 設立拠出金" と記載
- 通帳/取引明細を PDF 保存 (登記申請で添付)

### ④ 登記申請 (登記ねっと、 法務省オンライン)

`https://www.touki-kyoutaku-online.moj.go.jp/` (登記・供託オンライン申請システム = 登記ねっと) を使用:

1. **申請者情報登録** (初回のみ) — マイナンバーカードで本人確認
2. **設立登記の申請書を作成** — システム内で対話的に入力
3. **添付書類を電子化**:
   - 電子認証済み定款 (②で取得した PDF)
   - 基本金払込証明書 (③の通帳 PDF + テンプレート文書 PDF)
   - 設立時理事・評議員の就任承諾書 — 全員のマイナンバーカード電子署名
   - 印鑑届書 — **代表者個人実印 を電子証明書 (マイナンバーカード) で代替可能**
4. **登録免許税 ¥60,000** を電子納付 (ペイジー / インターネットバンキング)
5. **電子署名して送信** — マイナンバーカードで申請書全体に署名

通常 **5-10 営業日** で登記完了。完了通知は登記ねっとのダッシュボードで確認 (メール通知も来る)。

### 設立後

- 登記完了後 14 日以内に **税務署 / 都道府県 / 市町村** に法人設立届書を **e-Tax / eLTAX** で電子提出
- **法人実印は不要** (電子証明書で代替)
- **銀行口座開設**: マイナンバーカード + 履歴事項全部証明書 (登記ねっとから PDF 取得可) で 100% オンライン申請できる銀行を選ぶ (GMO あおぞらネット銀行 / みんなの銀行 / 楽天銀行 等)

## 2 年後 → 公益財団法人への移行

一般財団法人として 2 期 (約 2 年) 運営した後、 公益認定申請:

- 提出先: 内閣府 (主たる事業が全国又は 2 都道府県以上に及ぶ場合) / 都道府県知事 (1 都道府県内のみの場合)
- **公益認定等委員会** での審査
- 申請も **e-Gov** から電子申請可能

公益認定後の追加効果:
- みなし寄付金 (収益事業から公益目的事業への支出を全額損金算入)
- 寄付者の所得税控除 (年間所得の 40% まで)
- 法人寄付者の損金算入 + 特別損金算入枠拡大
- 名誉性が「Yuki 個人事業」から「公益団体」に変わる (§29 と整合)
