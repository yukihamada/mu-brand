## 4. コンバージョン計測（必須・無料登録の追跡）

### Method A: Google Tag Manager で /you の subscribe 完了を計測

`/you` ページ送信後の遷移先 or サンクスメッセージ表示時に dataLayer push:

```html
<script>
  window.dataLayer = window.dataLayer || [];
  window.dataLayer.push({event: 'you_subscribe'});
</script>
```

GA4 で `you_subscribe` イベントをコンバージョンに昇格させ、Google Ads で「インポート」して計測。

### Method B: Google Ads タグ直接埋め込み

Google Ads → ツール → コンバージョン → 新規 → ウェブサイト → イベントスニペット を取得し、`/you` 完了画面に埋める。
