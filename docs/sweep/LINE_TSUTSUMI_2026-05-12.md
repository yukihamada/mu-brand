2. 何ができるか
────────────────────

────────────────────
3. サンプル価格のルール
────────────────────
・**Printful 仕入原価のみ** で販売します。MU 側の利益はゼロ。
・サンプル目的の発注なので、通常販売とは別扱い（Stripe 決済 → Printful へ自動 draft order 投入）
・配送先は決済時に Stripe Checkout で入力（日本国内のみ）
・決済後の流れ：
　 - Printful 自動製造分 → 即 draft order → 製造・発送（10〜14日）
　 - 手動生産分 → 濱田まで Telegram 通知 → 個別対応
・原価未登録の商品はサンプル不可（ボタンが押せない仕様）
