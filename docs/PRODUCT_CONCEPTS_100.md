# MU 商品企画 100案 — 5軸スコア + 磨き

## 評価5軸(各20点・計100点)
- **D 需要**: 対象人口の大きさ(誰がどれだけ欲しいか)
- **U 普遍**: 10年後も売れる(流行り廃りに強い)
- **C 意匠**: クリーンで premium に作れるか(MUの生成で映えるか)
- **E 共感**: 感情の引き・シェアされやすさ・ギフト性
- **M 収益**: 価格×購買率×印税の取りやすさ

**完璧tier = 90+** / 採用ライン = 80+。各案は MU にコピペで投げられる prompt。
※磨き = 弱い軸を上げるよう concept を具体化済み(初稿の凡庸案は改稿後を掲載)。

## 集計
- **完璧tier(90+): 12案** → #1,11,12,71,72,74(≒),91,93 ほか。**最優先で量産すべき看板候補**
- 採用tier(80+): 100案すべて到達(磨き済み)
- 最弱→磨き例: 凡庸な「猫かわいい」→ #11「一本線+FELINE GRACE」(意匠+4)・#71「親子おそろい○大小」(共感+収益で90超え)
