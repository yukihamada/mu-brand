# MSA Charter Member Outreach — First 5 (and how to get quote-worthy buyers)

**Status:** Draft (2026-05-20) — Yuki が 個別 DM 送信
**Trigger:** Tanaka FB #10 — 「buyer の voice が ない、 既購入者 quote 1個 あれば 今晩 ポチる」

## 何 を 解決 する か

- /source ページ に **「実在 する 既購入者」** の quote が 0個。 信頼性 が 仮説 のまま。
- 一般 公開 メール blast / X thread に 1人 の **実体験 が ない** → 「初手 で 買う 勇気 が 出ない」 が 起きる。

## 戦略

**First 100 全員 を セールス しない**。 まず **5名 を 個別 で クローズ する**。 5名 が 入って quote を 出して くれた 時点 で、 LP に 載せて 残り 95名 を 一般公開 で 売る。

「**charter member (発起会員)**」 と 呼ぶ。 5名 だけ の 特別 ステータス を 与える。

## charter member の 条件

charter には:
- ¥9,800 bundle (案B 採用 前提) → **¥4,900 で 提供** (50% off)
- MSA lifetime + 「charter member ロール」 (Discord/Telegram の Pinned 名前 + /source ページ 末尾 の 「founding members」 セクション に 名前 or alias)
- 月1 30分 Yuki と オンライン Q&A (Phase 2 Founder tier の preview)
- 引き換え に: **Phase 2 で zip を pull した あと、 一行 で 感想 を 公開 用 に 提供**

5名 だけ。 期間限定。 締切 = 2026-05-25 (5日後)。

## 送信 ルール

1. **個人 ごと に 必ず ハンド トランスレート**。 テンプレ 丸コピ NG。 hook 1-2行 を 入れて 「君 にだけ」 感 を 出す。
2. **1日 1通**。 5日 で 5枠。 急がない。
3. **断られたら 引きさがる**。 「読んで くれ」 と 食い下がる の は **逆効果**。 「またね」 で 終わる。
4. **承諾 もらったら 個別 Stripe Payment Link を 発行**。 全体 商品 ページ から 買わせない (charter 専用 50% off を 別 SKU に する)。
5. **入金後 24h 以内 に Yuki から DM で 「ありがとう」 + Discord/Telegram グループ 招待 リンク**。

## charter quote の 取り方

入金 → Tシャツ 発送 → MSA Phase 2 で trio リポ を pull して もらう → **2-3日 後 に 質問 1問** だけ:

> 「**今回 MSA で 読んだ コードで、 一番 意外 だった / 役に立った の は どこ ですか? 1-2行 でいいです**」

返事 を そのまま (or 軽い 編集 + 本人 OK 取って) /source ページ の 「what charter members said」 セクション に 掲載。

**やらない こと:**
- お決まり の 「最高でした!」 系 を 書かせる
- 文章 を こちら で ライティング する
- 賞賛 だけ を 集める (批判 quote も 載せる。 真実 性 が 信用 を 上げる)

## 締切 後 の リカバリ

5/25 までに 5名 集まらなかった 場合:

- 集まった 人数 で start (3名 でも 公開 OK)
- 残り 枠 を 一般 公開 charter として LP に 出す (「あと N 枠」 表示)
- 締切 を 1週間 延長 する **な**。 「期限 守らない 主宰」 は 信頼 を 失う

## /source ページ 反映

charter quote が 揃ったら /source ページ の section 構成 に:

```
<h2>charter members が 言って くれた こと</h2>
<blockquote>
  「{quote}」 — {name or alias} (charter #{n})
</blockquote>
... (5個)
```

を 追加。 顔写真 や 実名 は **本人 の 明示 同意 が あれば のみ**。 alias で OK。
