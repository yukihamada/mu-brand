軽さ/肌当たり着た瞬間に分かる差（体験価値）
サステナカットロス0・縫いしろ0 の端材ゼロ生産
1型を磨く反復生産で型費（編成プログラム）を償却 ＝ 1着あたりは下がる

→ 万人向けの安価Tではなく「価値が分かる人」向けのプレミアム1型として売る。
