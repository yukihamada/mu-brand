## 0. なぜホールガーメントか(WG採用理由)

縫製カットソーより高い。それでもWGにする理由を価格で説明できる形に:

- **縫い目ゼロ** → 脇/肩がねじれない・擦れない・破れの起点が無い = **一生もの級の寿命**
- **軽い・肌当たり** → 着た瞬間に分かる差(=体験価値。ECで伝わりにくいので文章と動画で補う)
- **カットロス0・縫いしろ0** → 端材ゼロのサステナブル生産(物語になる)
- 型は**1型を磨き切って長く売る**(憲章) → 型費(プログラム)を長期で償却 = 1着あたりは下がる

→ ゆえに**万人向けの安価Tではなく、価値が分かる少数向けのプレミアム1型**として売る。

## 6. 仕上げ / 縮率 / 表示(v2 追加)

- **仕上げ**: 洗い + プレス(仕様は工場標準で可)。**縦・横の収縮率(%)を試編み時に申告**してもらう。
- **品質表示タグ**(家庭用品品質表示法): 組成「綿100%」・原産国・取扱絵表示。後付け縫製1点のみ。
- **洗濯表示**: 家庭洗濯可(ネット使用)を目標 → 洗濯試験で確定。
- 編みネーム/織ネーム(MU)を後ろ首に1点。

## 9. フェーズ計画(型費償却＋小ロット現実対応)

1. **Phase 1 — 試編み1着**(きなり / サイズ2(M) / 18G / 綿100)。型確定・目付/縮率確認・実物撮影。
2. **Phase 2 — 量産 3サイズ**(きなり)。Phase1の型を反復し型費を償却。実物写真でPDP差し替え。
3. **Phase 3 — 色/素材拡張**(染め糸MOQ・メリノ混の見積り後)。

> 価格(retail)は Phase 1 の実原価(型費按分＋糸＋編成)が出てから確定。
> それまで公開面に価格を出さない / go-live しない。
