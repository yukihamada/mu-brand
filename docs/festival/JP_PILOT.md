# 無音祭 ZERO — 日本プレイベント企画
### ハワイ本番(2026.10.29)の前に、東京で小さく一度まわす

> 目的は「集客」ではなく「**実証**」。
> 無音祭OS（サイレントディスコ・T=入場・無人運営・フェイルセーフ）が
> 本当に動くかを、低リスク・小規模・身内中心で1回確かめる。
> ここで壊して、ハワイで壊さない。

---

## 0. なぜ先に日本でやるか（3つの理由）

1. **リハーサルになる** — Web Audio 同期、AI DJ、キルスイッチ、避難導線を本番前に実地で検証。
2. **失敗が安い** — 身内30〜80人・屋内・短時間なら、何が起きても取り返せる。ハワイは一発勝負。
3. **物語が貯まる** — 「#0 無音祭」を撮っておけば、ハワイ告知の最高の素材になる。MUの世界観を見せられる。

キーワード：**ZERO（0号）= 無の祭の、最初の一回。**

## 1. コンセプト

**「無音祭 ZERO — 30分だけ、東京を無音にする」**

- 全員ヘッドホン（サイレントディスコ）。外から見ると無音の群衆、中では音が鳴っている。
- 入場は **MUGENのTシャツ**（=ハワイと同じルール。ここで運用を試す）。
- 運営は最小。音・進行・配信はAIが回し、人間は「止める/解禁する」だけ。
- クライマックスは **日没（または会場の照明を一斉に落とす瞬間）に1曲だけ全員同期** → ハワイのサンセット解禁の予行。

## 2. 規模・場所・時間（最小実証セット）

| 項目 | 推奨デフォルト | 理由 |
|------|--------------|------|
| 人数 | **30〜80人**（招待制） | 身内＋MUer。事故っても収拾できる規模 |
| 会場 | 都内の小箱／スタジオ／ギャラリー／柔術ジム夜間 | 防音不要(無音だから)・電源・WiFiだけ要る |
| 時間 | **平日夜 90分**（うち本番30分） | 短く濃く。撤収まで含めて軽い |
| 費用感 | 会場代＋飲み物のみ（出演料0・PA0・チケット印刷0） | 無人設計だから固定費が消える |
| 入場 | MUGEN T 着用（無い人は当日その場で号外を買う動線） | ハワイのチケット運用テスト |

> 候補会場の当たり：北参道SIIIEEP（BJJ枠と相性）/ 焼肉古今の貸切夜 / 都内スタジオ。
> ※ どれも要・先方確認。公開コピーにSIIIEEPは出さない（契約待ち）。

## 3. 当日タイムライン（90分・ほぼ無人進行）

```
T-30  開場。QR/Tシャツで入場（人間は1人、流れを見るだけ）
       各自スマホで /fest/sync に接続 → ヘッドホン装着
T-0   無音開始。AI DJ がチャンネル3つ（無/間/無限）を配信
 +10  「心拍チャンネル」起動：会場の平均心拍がBPMになる（②の実証）
 +20  柔術ロール in サイレンス（有資格者1名、観るのは自由）（⑤）
 +28  全照明を落とす → 30秒の沈黙
 +29  【クライマックス】全端末が1曲だけ完全同期で一斉発音（③④の核）
 +30  解禁の合図 → ハワイ座標ティザー or 次の告知を投影
 +30〜 自由歓談（音は流したまま）、撤収
```

人間の操作はたった2回：**開場の「始める」** と、クライマックスの **「解禁する」**。あとは全部自動。

## 4. これで"何を実証するか"（チェックリスト）

| 検証項目 | 成功条件 | ハワイへの学び |
|---------|---------|--------------|
| Web Audio 同期 | 30〜80台が±50ms以内で一斉発音 | ③の心臓部。スケール前提を確認 |
| T=入場 運用 | 行列なく流れる／無い人が号外を買える | ハワイのゲート設計 |
| AI DJ | 承認済みセットだけ・無事故で90分 | ①の安全フロー |
| キルスイッチ | 1ボタンで全端末が即無音 | 絶対安全の最終手段 |
| データ最小 | 心拍/声を保存ゼロで運用 | 信頼＝ブランド |
| オフライン耐性 | WiFiを意図的に切っても裏セットが鳴る | 会場回線が死んでも止まらない |

→ ここで通れば、ハワイは「同じものを大きくするだけ」になる。

## 5. 最小スタッフ（コア2〜3名）

- **あなた**：始める/解禁する判断（人間ゲート）
- **監視1名**：スマホ/ノートでダッシュボードを見る、赤が出たら対応
- **柔術枠の時だけ有資格者1名**

※ 受付・音響・照明・もぎりは全部消える（無音＋デジタル＋T入場のため）。

---

## 6. 絶対安全（小規模版フェイルセーフ）

1. **キルスイッチ** = 手元の1ボタン → 全端末無音＋場内灯点灯。
2. **音量ハードキャップ** = クライアント側で上限固定（ヘッドホン難聴防止）。
3. **避難導線** = 屋内なので出口確認だけ。事前に1枚図を貼る。
4. **データ破棄** = 心拍・声はその場で平均化して即消去、保存しない。
5. **決済は事前/その場のオンラインのみ** = 現金を扱わない。
6. **少人数 = そもそも低リスク**。これが最大の安全装置。

---

## 7. 実装の最小単位（ここから作る）

ZERO のために本当に要るのは2つだけ：

1. **`/fest/sync`（noindex）** — スマホで開く同期プレイヤー。
   サーバー時刻に合わせて、指定時刻に一斉発音。音量上限つき。裏セット内蔵。
   → これが③④の核であり、ハワイでもそのまま使う。
2. **オペレーター画面 `/fest/console?token=`** — 「始める」「解禁する」「全停止」の3ボタンだけ。
   → 人間ゲートの実体。

その他（心拍②・声⑦・3ch⑥・AI DJ①）は ZERO で“デモ版”として薄く乗せ、ハワイへ向けて育てる。
