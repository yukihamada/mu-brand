## 確認すべき項目(問い合わせで必ず聞く)

製品仕様は v2(`seamless_knit/SPEC_MU_SEAMLESS_ONE.md` / `tech_pack.json` / 見やすい版 `index.html`)。
**18G・コットン100・きなり1色・Phase1は試編み1着**という前提で聞く:

1. **最小ロット**(まず**試編み1着**から可能か)
2. **18G 対応可否**(薄手Tシャツ的なハイゲージが編めるか)
3. **在庫コットン糸**(18Gに合う在庫糸があるか=専用糸MOQを避けたい)
4. **型費(編成プログラム制作費)/型** と **サンプル代**
5. **量産単価**(数量帯ごと)・**リードタイム**(サンプル/量産)
6. **縮率(%)**(縦横の収縮率を試編みで申告してもらえるか)
7. **データ入稿形式**(本仕様書+図で足りるか/APEX データ要否)
8. (将来)染め糸・メリノ混の **糸MOQ**(色/素材拡張=Phase 3 の判断材料)
