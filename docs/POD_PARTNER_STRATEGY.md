# MU 直接提携・AIネイティブPOD化 戦略

> 結論: MUは「もう一つのPOD(印刷の安売り)」になってはいけない。
> **意匠・需要・顧客・課金を握り、印刷は黒子の国内工場に“直接”外注する
> = AIネイティブPOD** になる。これが「他のPODみたいに直接パートナーになる」の
> 正しい形であり、唯一勝てる形。

## 1. 核心テーゼ — なぜ「普通のPOD」では負けるか

| | 既存POD(Printful/SUZURI/Gelato) | MUが持つもの |
|---|---|---|
| 売り物 | 印刷・物流(供給) | **需要と意匠(言えば作れる)** |
| SKU | 利用者が絵を上げる | **言葉から無限に生成** |
| 供給 | 単発 | **印税でcreatorがUGCとして量産＋宣伝** |
| 顧客 | EC事業者 | **作り手＝買い手＝広める人(community)** |

印刷は誰でも買えるコモディティ。供給で戦うとマージン争いで負ける。
MUは**需要・意匠・顧客・課金の上流**を握っている。だからMUは
「印刷屋になる」のでなく、**印刷を黒子に回し、上流を独占する**。

## 2. 2フェーズ戦略

### フェーズ1 — 工場と「直接提携」(POD転売をやめ一次生産者になる)
- 現状: Printful(海外転売)経由 → 利ざや薄・海外・品質/納期の制御が弱い。
- 次: **国内工場とB2B/API直契約**。MU=ブランド/プラットフォーム、工場=黒子。
- 候補(調査済・1着からAPI・国内製造):
  - **Printio(OpenFactory)** ⭐ — API(発注/製造ステータス/追跡)・1点・全国工場・事業者向け。hello@printio.me / printio.me/printioapi
  - **Gelato(日本生産)** ⭐ — REST API強い・**MUに `gelato_jp` ルート既存**=接続最短
  - イメージ・マジック / UP-T(母体OEM) — 要問い合わせ
- 効果: ①POD利ざやを取り戻す ②**Made in Japan/速い納期**の物語 ③品質・体験の制御 ④BJJ垂直で「国産チームギア」が売れる

### フェーズ2 — MU自体がPOD/創作プラットフォーム(他者がMUの上に作る)
- MUの「**言えば作れる + 着画 + 印税 + fulfillment**」を **API/ホワイトラベル**で開放。
- 顧客: 他ブランド/アプリ/クリエイター/**道場**。彼らはMU経由で
  「AIで意匠を生む＋在庫ゼロで売る＋印税でcreatorを巻き込む」を一括で得る。
- 差別化: 既存POD=「あなたが絵を上げる」。MU=「**言葉から絵を生む＋maker経済つき**」。
- 土台は既にある: **MCP / `/api/agent/*` / `/api/app/agent/chat` / make / 印税**。
  → 「MUのAPIを叩けば、誰でもAIネイティブPODを自社に組み込める」。

## 3. 経済 — なぜ直接提携が効くか

```
POD転売:   売価 − (工場原価 + POD利ざや) − 印税 = MU取り分(薄い)
直接提携:   売価 −  工場原価            − 印税 = MU取り分(POD利ざや分が戻る)
```

戻ったマージンを **①印税原資(10→最大50%)** と **②広告/CAC** に回せる。
= maker経済(供給)と顧客獲得(需要)の両輪が同時に強くなる正のループ。

## 4. BJJ垂直で先に証明(mu-brand戦略との整合)

- 第一弾: **道場公式ギア(ラッシュガード/帯モチーフ)を国内工場直**で。
  「Made in Japanの道場ギアが、言うだけで作れて、売れたら道場に印税」。
- ここで工場直の品質・納期・原価を実証 → 一般アパレルへ展開。
- JiuFlow(実需)× MU(意匠/maker経済)× 国内工場直 = 三位一体。

## 5. 直近アクション(人間ゲート明記)

1. **Printio + Gelato-JP に問い合わせ**(API仕様・最低ロット・原価表・納期・
   技適/法規不要グッズ範囲)。→ 一次接触メール下書きはAIが作成。
2. **1社でMVP**: 1 kind(例: tee or ラッシュガード)をMUの `fulfillment_route`
   に直結(`gelato_jp` は半分実装済 → ここが最短)。
3. **原価/マージン試算** → 印税レンジと売価を再設計。
4. ⚠ 有料コミット(最低発注・月額・サンプル¥30,000超)は **本人承認**(BUDGET.md §3)。

## 6. やらないこと(明確な非戦略)

- 印刷の安売り競争(commodity)に参入しない。
- 自社工場/在庫を持たない(無在庫の原則は維持)。
- 海外量産で「Made in Japan/言葉が形になる物語」を捨てない。
- フェーズ2(プラットフォーム開放)を、フェーズ1(直提携で品質実証)より先に出さない。
