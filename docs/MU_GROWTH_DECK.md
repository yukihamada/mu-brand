## 2. 3-layer retail thesis

| layer | 何 が 動く 市場 か | 代表 | 規模 (Japan) |
|---|---|---|---|
| **既存品** | 既に 工場 で 作られた 物 | Amazon Japan | **¥6T**/年 (2023) |
| **中古品** | 個人 が 既に 所有 してる 物 | Mercari | **¥1T**/年 GMV (FY2024) |
| **未存 品** | **まだ 存在 しない 物** | **MU** | **¥500B+**/年 (TAM 計算 下記) |

「すでに 存在 する 物」 → 「すでに 持ってる 物」 → **「まだ 存在 しない 物」**。 これ が 次 の retail layer。

## 3. TAM 計算 (Japan のみ)

### 3-A. potential creator pool

| 集団 | 人数 | source |
|---|---|---|
| TikTok JP active creator | **1.7M** | TikTok 公式 2024 |
| Instagram JP creator (フォロワー 1K+) | ~**2.5M** | Meta 2024 推定 |
| 小規模 事業者 + フリーランス | **3.5M** | 国税庁 申告者 数 |
| 趣味 community (BJJ, surf, etc.) active | **~10M** | 各 連盟 + アンケート |
| **重複 除く 推定 creator pool** | **~5-8M** | |

### 3-B. ARPU 想定

| tier | 月間 売上 / 人 | 該当 % |
|---|---|---|
| 趣味 (たまに 1 着) | ¥1,000 | 70% |
| 熱量 ある creator | ¥10,000 | 25% |
| 道場 / 店舗 / power user | ¥100,000 | 5% |

加重 平均 ARPU = ¥7,700/月/active user

### 3-C. TAM

- conservative: 5M creator × 1% adoption × ¥7,700/月 × 12 = **¥4.6B**/年
- base case: 5M × 5% × ¥7,700 × 12 = **¥23B**/年
- bull case: 8M × 10% × ¥7,700 × 12 = **¥73B**/年

→ **Japan だけ で ¥4-73B**。 global × 30-50 倍 = **$1-15B TAM**。

## 7. 数字 で 見る 「世に出す」 圧縮

### A. 個人 → 物 → 売上 の 時間

```
従来:  アイデア ─── 5 ヶ月 ─── 売上 1 件目
MU:    アイデア ── 90 秒 ── 売上 1 件目
       compression: 5 ヶ月 ÷ 90 秒 = 約 144,000x
```

### B. 個人 → 物 → 売上 の 投下 金額

```
従来:  ¥500,000 投下 → 50-100 着 売れて break-even
MU:    ¥0 投下 → 1 着 売れて break-even
       leverage: 無限 (cost ÷ 0 = ∞)
```

### C. SKU 生成 能力 (上限)

```
従来 個人: 年 1-3 デザイン (時間 制約)
MU: 24 デザイン/日 × 365 = 8,760 デザイン/年/AI サイクル
    × 108 着 (MUGEN cycle 上限) = 946,080 着 / 年 / AI サイクル
    ARPU ¥5,000 × 10% sell-through = ¥473M / 年 / AI サイクル の 上限
```

= 1 AI サイクル だけ で **年 約 ¥473M** の 上限。 24 cycle (時間 別) 並列 で **¥11.3B** の 理論 ceiling。

## 8. 動詞化 戦略 (= Mercari メルカる の 再現)

| 過去 | 動詞 | 浸透 |
|---|---|---|
| Mercari (2014) | 「メルカる」 | 2018 で 国民 動詞 化 |
| MU (2026-) | 「MU する」 / 「MUって」 | 2028 で 同 ターゲット |

定義:
- 「MU する」 = アイデア を 1 行 で 物 に して 売る
- 「MU された」 = 誰か の 1 行 が 物 と して 出回って 売れた
- 「MU された 服」 = 個人 が 自分 で 出した T シャツ

→ Year 2 末 に 「MU する」 が Tech Twitter で **使われ 始める** こと が KPI。 Year 5 で **一般 に 浸透**。

| layer | moat | 競合 が 模倣 可能 か |
|---|---|---|
| POD pipeline | Printful + SUZURI 統合 | ❌ 模倣 可能 (誰 でも 同じ API) |

| **日本 craft narrative** | 弟子屈 + Loopwheel + 鉱物染料 | ⭕ 物理 拠点 + 工房 関係 |
| **multi-niche tribe data** | BJJ 170 道場 + 釣り + dev + ... | ⭕ 各 tribe の trust |
| **動詞化** (Year 2-5) | 「MU する」 が 一般 動詞 化 | ⭕ network effect |

→ **moat の 重心 = 系譜 + tribe trust + 動詞**。 これ ら は 3 年 で 累積、 競合 が 後 から 追えない。
