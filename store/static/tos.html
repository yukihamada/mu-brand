利用規約

最終更新: 2026年5月14日 / 制定: 2026年5月14日

本規約は、 株式会社イネブラ（以下「当社」）が運営する MU および wearmu.com（以下「本サービス」）の利用条件を定めるものです。 本サービスを利用することにより、 利用者は本規約に同意したものとみなします。

第1条 適用範囲

本規約は wearmu.com および付随するすべての機能（商品購入、 collab page 作成サービス、 MU credit、 サブスクリプション、 ニュースレター、 API 等）に適用されます。

第2条 アカウント

  1. アカウントはメールアドレスによる本人確認の上で作成されます。 1 人につき 1 アカウントとし、 虚偽の情報での登録を禁じます。
  2. セッション・認証情報の管理責任は利用者にあります。 第三者による不正利用が発生した場合、 当社は責任を負いません。
  3. 利用者はいつでも自分のアカウントを削除できます。 削除方法はプライバシーポリシー第 5 条をご参照ください。

第3条 商品の購入

  1. 商品はすべて受注生産（オンデマンド製造）です。 注文確定後のキャンセル・返品・交換はお受けできません（不良品・誤配送を除く）。
  2. 価格・配送・支払い等の詳細は 特定商取引法に基づく表記 に従います。
  3. 商品到着後 7 日以内に不良品・誤配送をご連絡ください。 mail@yukihamada.jp で対応します。

第4条 collab page 作成サービス

  1. creator（作成者）の責任: collab page に記載される内容（イベント情報・出演者・第三者の肖像・商標・著作物の利用許諾等）の真実性および適法性は、 すべて creator が責任を負います。 当社は事前審査を行いません。
  2. 当社の役割: 当社は collab 作成のためのプラットフォーム（chat UI、 デザイン生成、 Stripe 連携、 製造発注）を提供するに留まります。 第三者の権利侵害に関する紛争は、 一次的には creator と権利者の間で解決していただきます。
  3. 削除権: 法令違反・第三者の権利侵害・公序良俗違反の通報を受けた場合、 当社は creator への事前通知なく当該 collab page を非公開化できます。
  4. 無料枠: 認証メール 1 件につき 1 collab page まで無料。 また Opus チャットの利用枠として ¥20,000 相当の API 利用権を付与します。

第4条の2 投稿デザインとリミックス

  1. 著作権の帰属: 本サービス（/make 等）で利用者の指示により生成・投稿されたデザイン（以下「投稿デザイン」）に関する権利は、 法令上認められる範囲で投稿した利用者（クリエイター）に帰属します。
  2. 当社への利用許諾: クリエイターは当社に対し、 投稿デザインを商品の製造・販売・宣伝のために利用する、 非独占的・世界的・無償の許諾を与えるものとします。
  3. リミックスの許諾: クリエイターは、 他の利用者が投稿デザインを基にした二次的著作物（リミックス）を本サービス上で作成・販売することを許諾するものとします。 リミックス商品が販売された場合、 当社は元デザインのクリエイターに対し、 販売価格の 5% を MU credit として還元します（リミックスを行った利用者への還元とは別枠）。
  4. オプトアウト: クリエイターは、 第11条の窓口への申し出により、 自己の投稿デザインに対するリミックスの許諾をいつでも将来に向かって撤回できます。 撤回後、 当社は当該デザインのリミックス機能を無効化します（撤回前に作成されたリミックス商品には影響しません）。
  5. 適用: 本条のうちリミックスに関する規定は 2026 年 6 月 12 日以降の投稿デザインに適用されます。 それ以前の投稿デザインについては、 当社からの個別の告知に対して異議のなかったものに適用されます。

第5条 MU credit（社内ポイント）

  1. MU credit は本サービス内でのみ使用可能な社内ポイントであり、 1 MU = ¥1,000 として collab 作成費用等に充当できます。
  2. 付与額は商品購入額の 20% を上限とします（景品表示法 第 2 条第 3 項 一般懸賞・総付景品 規制に準拠）。 1 商品あたりの上限は ¥1,000 です。
  3. MU credit は 現金との換金不可・第三者への譲渡不可です。
  4. アカウント削除時、 未使用残高は失効します。
  5. 残高は当社の判断により予告なく仕様変更する場合があります（ただし既存残高は維持します）。

第6条 サブスクリプション（MU Drop Creator 980円/月）

  1. 月額 ¥980 (税込) の自動継続課金です。 解約はいつでも可能です。 解約後は当該課金月末まで利用できます。
  2. 日割り計算による返金は行いません。
  3. 支払い失敗が 7 日続いた場合、 サブスクリプションは停止されます。

第7条 禁止事項

  • 法令、 公序良俗、 第三者の権利を侵害する行為
  • 他人になりすますこと、 虚偽情報の登録
  • collab page を利用したマネーロンダリング・脱税・詐欺・寄付詐称
  • 差別・誹謗中傷・暴力・性的搾取・テロ・違法薬物関連のコンテンツ作成
  • 未成年者を被写体とする不適切なコンテンツ
  • 当社・本サービスへの DoS / 自動スパム / 不正アクセス
  • 本サービスのリバースエンジニアリング、 認証情報の共有

第8条 免責

  1. 当社は本サービスの完全性・正確性・有用性を保証しません。 AI が生成するデザイン・テキストには誤りが含まれる可能性があります。
  2. 当社は天災・通信障害・第三者サービス（Stripe / Printful / SUZURI / Resend / Cloudflare / Anthropic 等）の停止に起因する損害について責任を負いません。
  3. 当社の損害賠償責任は、 当該利用者が直近 3 ヶ月に当社へ支払った金額を上限とします（当社の故意・重過失の場合を除く）。

第9条 規約の変更

当社は本規約を予告なく変更できます。 重大な変更時は本ページ更新および登録メールにて事前に通知します。 通知後の継続利用は変更への同意とみなします。

第10条 準拠法・裁判管轄

本規約は日本法に準拠します。 本規約に関する紛争は東京地方裁判所を第一審の専属的合意管轄とします。
