FAQ

Q. NFT や ウォレット は 要る？

要らない。 Stripe checkout の メール だけ で 照合。 magic-link ログイン で 完結。

Q. 返金 したら ソース は どうなる？

返金 した 時点 で 以降 の DL は 403。 すでに DL 済み の zip は 取り消せない (現実的に 不可能)。 そこは 性善説。
