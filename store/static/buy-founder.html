FAQ

¥48,000 は 高すぎる?

VISVIM SS Pocket Tee ¥30k 台、 CDG SHIRT Forever Tee ¥20k 台、 MA オークション ¥30k〜¥100k と 並ぶ tier。 「服 を 1着 100年 大事に する」 文化 を 物理化 した 価格。 ¥9,600/着 は 修繕 reserve fund に escrow。

「100年 保証」 は 法人 が 倒れたら?

tsugi succession protocol で 後継法人 が 引継ぎ。 売却・破産時 は 鉱物染料 配合 と 縫製図 を 全 owner に 開示 (self-修繕 可能)。 reserve fund は 信託 口座 (詳細 確定後 公開) で 法人 倒産時 も 保全。

NFC chip は 100年 持つ?

持たない (NFC chip MTBF ~10-20年)。 そのため 25 年 ごと に Enabler Inc. が 無料 交換。 chip 死亡時 も 革ラベル の 箔押し serial 番号 + on-chain mint で identity 復元 可能。 100年 narrative は chip ではなく serial + anchor で 担保 する。

海外 から 買える?

可。 DHL international 配送、 関税別。 ただし 鉱物染料 5g は 国際郵便 危険物 規制 (UN3077) で 同梱できない 場合あり。 海外 buyer に は 修繕リクエスト 時 別送 (or Enabler 担当者 持参) で 対応。

Solana wallet 持ってない けど DAO seat 行使 できる?

可。 Phantom (Chrome 拡張) セットアップ ガイド を Chronicle ページ に 添付。 wallet 未保有 buyer は メール vote 代行 mode も 用意 (= 法人 が wallet 管理 して 投票)。 後日 self-custody へ 移行 可能。

誰 が 作るの? Yuki 1人?

#001-#100 (初期 25年) は 創業者 (濱田 優貴) 直接 関与 (手書き ノート + 制作 立会)。 #101 以降 は 後継 founder が 連名 で 引継ぎ (tsugi succession token)。 物理 製造 は 和歌山 Loopwheel 工房 + 弟子屈 染色 + DTG 印刷者 の 三者。

返品 / キャンセル?

製造開始前 = キャンセル 全額返金可。 製造開始後 = サイズ違い / 配送事故 のみ 全額返金。 デザイン 不満 で の 製造後 キャンセル は 不可 (1着 限定 製造 のため)。 NFT 部分 は 二次流通 で 物セット 売却 可能。

領収書?

Stripe receipt + Enabler Inc. 発行 領収書。 適格請求書発行事業者番号は 登録完了後 にメール領収書 に 記載 (お申込み時点 では 通常の 領収書 で 発行)。
