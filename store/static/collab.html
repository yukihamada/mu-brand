料金を読む前に、7 つの言葉だけ揃えておく。

SKU

商品 1 種類のこと

Stock Keeping Unit。 "T シャツ・黒・M サイズ"が 1 SKU。 「30 SKU 作る」= 30 種類の商品ページが立ち上がる。

PT

MU 内クレジット

1 PT = ¥1 等価。 SKU 生成 (10 PT 消費) と wearmu サンプル購入に使える。 残高は /collab/chat の右上に常時表示。

RS

レベニューシェア

売上 (Printful 仕入 + Stripe 控除後) から MU が受け取る取り分。 標準 30%、 MU 保有で最安 5% まで段階で下がる。

MUGEN

連番刺繍 T (¥38,800)

MU の象徴的 1 着、 1 着ごとに固有の連番が襟裏に刺繍。 購入で +1000 PT bonus、 RS が 15% に下がる。 wearmu.com で購入可。

MUON

白帯〜黒帯 (¥6,800)

柔術帯モチーフのアパレル。 保有で RS 20% に下がる。 PT bonus は無し、 でも入場ゲートを開ける "鍵" として機能。

MA

MU Asset NFT

MU の 100 限定 NFT。 保有で RS 10% & 1 生成ごと +3 PT bonus (= ¥7/SKU 実質)。 100 日に 1 回 founder relay で次の保有者に贈与される連鎖式。

Constitution

無料 NFT (誰でも mint 可)

MU の理念に賛同する人が mint する NFT。 保有だけで /collab 入場可能、 ただし RS は標準 30%。 「MU の市民権」の最小単位。

よくある質問

導入前に 必ず聞かれる 9 つの質問。

本当に基本料 ¥0？仕掛けは？
仕掛け無し。¥10/SKU の従量課金 + レベシェアだけ取ります。 月額固定費はゼロ、契約縛りもなし。アクセスには MUGEN / MUON / MA いずれか 1 つの保有が必要で、これが実質的な「入場料」を兼ねます。MUGEN は ¥38,800 (1 着で +1000 PT bonus = 100 SKU 分が即返ってくる)、MA は NFT 保有で permanent。
AI デザイン枠 / Printful 仕入 / Stripe 手数料は売上から実費で引かれます (利益にはダメージ無し)。
PT って何？どうやって貯まる、どう使う？
1 PT = ¥1 等価の MU 内クレジット。SKU 生成 (1 つ 10 PT 消費) と wearmu サンプル購入に使えます。
貯め方:
• wearmu でサンプル購入 → 売価の 10% を PT 還元 (¥4,800 T → 480 PT)
• MUGEN 1 着購入 → +1000 PT bonus (= 100 SKU 分)
• MA 保有 → 1 生成ごと +3 PT bonus (実質 30% 還元、¥10 → ¥7/SKU)
PT 残高は /collab/chat の右上に常時表示され、生成のたびに自動消費されます。
レベシェア 30%→10% の段階はどう決まる？
MU 保有状況で自動判定されます (Stripe / NFT ウォレットを binding 済みのメールから探索)。
Free (Constitution / 過去購入のみ): 30% · MUON: 20% · MUGEN: 15% · MA: 10% · MUGEN + MA stack: 5%。
MUGEN/MA を後から取得すれば次の売上から自動でレートが下がります。再申告不要。
デザインの最終承認は誰がやる？
あなた。 AI が生成した SKU はあなたの draft preview に並び、1 つずつ「公開 / 非公開」をボタンで判定できます。NG が出続ければプロンプトを調整し、好みのトンマナに収束します。既存パートナーの場合、第二弾以降は AI の通過率 90%+ に到達しています。
Printful の品質は？返品クレームは？
Printful は北米最大の on-demand 印刷ファクトリー (年間 数千万 SKU)。当 31 SKU すべて API レベルで draft 注文 → 検証済。
返品 / 印刷ミスは Printful が 100% 無料再印刷。MU の自律エージェント (auto_refund) が ¥10,000 以下の苦情を自動分類 → Stripe で即返金 → Printful に再発注の流れまで自動化済。
既存の自社サイトに埋め込める？
1 行でできます。 <script src="https://wearmu.com/embed.js" data-brand="yourbrand" defer></script>
Notion / Wix / Webflow など JS が禁止のサイトには iframe 版もあり。SSR (Next.js / Rails / Python) 用に JSON API も公開済 (CORS 全オリジン許可)。
うちのロゴ / ブランド IP は守られる？
ロゴ画像は Cloudflare R2 (専用バケット) に格納、AI 学習データには使用しません。Enterprise プランでは NDA + SOC2 対応中の専属インスタンスを提供。
全ファイル削除を希望すれば GDPR 削除リクエスト 1 回でクリアします (自律エージェントの compliance_watch が滞留監視)。
売上の振込先は？
デフォルトは MU の Stripe アカウントに着金 → MU が月末締めでレベシェア分 (5〜30%、保有レベル次第) と SKU 生成費を差し引いて翌月末にパートナー口座へ振込。
MUGEN + MA stack (RS 5%) 以上の長期パートナーは あなたの Stripe アカウントに直接着金 → MU の取り分を翌月インボイスで請求 に切り替え可 (法人税務的にクリーン)。
解約は？年間契約縛り？
そもそも月額契約がありません。使った分だけ ¥10/SKU の従量課金なので、いつでも止められます。
MU 保有 (MUGEN/MUON/MA) は手放さない限りアクセスは維持されます。やめたければ生成を止めるだけ。
解約時はデザインデータ / 商品 DB / 顧客リスト すべて CSV / JSON でエクスポート可。lock-in しません。
暗号通貨決済 (ENAI / Solana) の手数料は？
Stripe の 3.6% に対して Solana 経由は 0.5% (Helius RPC + 自社 webhook)。ENAI トークン (Enabler ガバナンス) と USDC で受領可。Enterprise プランでは NFT ゲート販売 (例: 限定 100 体ホルダーだけが買える) もサポート。
競合 (Shopify / Printful 直 / Spring) との違いは？
Shopify はあくまで EC プラットフォーム。在庫 / デザイン / 物流は別途必要。月額 ¥3,500〜。
Printful 直 は製造のみ。デザイン・EC・CS は自前。
Spring (Spring.fly) はインフルエンサー向けでブランド統制が弱い。
MU は AI デザイン + 製造 + EC + CS + 法務まで一気通貫、しかも基本料 ¥0 + 従量 ¥10/SKU + RS 5〜30%。完全比較表は本ページ 料金セクションにあります。
