Verification

誰でも以下のコマンドで JSON のハッシュを確認できます: 

curl https://wearmu.com/will/v1.0.0-draft.json | \
  python3 -c 'import sys,json,hashlib;
d=json.load(sys.stdin);
c=json.dumps(d,ensure_ascii=False,sort_keys=True,separators=(",",":")).encode();
print(hashlib.sha256(c).hexdigest())'

# 出力が 071f3f77ba04b484478d67cd31032ee61d482b259f42c7927378354ad26add64
# と一致すれば、JSON は Solana に焼かれた時点から改ざんされていない

Bitcoin プルーフの検証: 

pip install opentimestamps-client
ots verify v1.0.0-draft.json.ots
# Bitcoin block での確定時刻を確認
運用方針:
本JSONは「テンプレ感のない付言事項」として運用しています。法的拘束力のある遺言事項は公正証書側に記載されています。改訂時は新バージョンを発行し、supersedes に旧バージョンの SHA-256 を記録、Solana に再アンカー、registry.json を更新します。
Constitution との関係:
MU の Constitution §28・§29 は生前から動作する事業構造です。本遺言は私の死後にそれを停止させないための後方支援文書です。
