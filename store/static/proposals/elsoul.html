1. なぜ MU からこの提案を — そして、 なぜ「社員もユーザーも欲しい」 のか

ELSOUL LABO B.V. 様の 「For Sustainable Innovation Environments」 は、 Solana 界隈における めずらしく深い思想です。 速い・安いだけでなく、 カーボン・電力・人 のレイヤーまで責任を取りにいく姿勢。 ERPC が 300+ edge を Green Web 認証で回し、 Epics DAO validator が Shinobi で全世界 3 位 — これは「結果として速い」 のではなく 「思想を技術にした結果速くなった」 順序です。

MU は AI が 1 着ずつデザインを生成する 0 人運営の自律 D2C ブランドです (Constitution + Vision: /constitution)。 利益の 50% を北海道弟子屈町に寄付する義務をコードに焼き込み済み (§27)。 御社と同じく 「持続可能性をハードコード」 という思想を、 ブランド側で実装しています。

本案件の特徴 — Dual-target

本 collab は「社員 (約 197 想定) + ユーザー (Solana 開発者・バリデーター・HFT トレーダー・dApp builder)」 両方が欲しがる SKU 構成です。 単なる corporate ノベルティではなく、 グローバルな Solana コミュニティに「持つこと自体が信頼の証」 になるアイテムを設計しました。

  • 社員向け (Inside-out): CREW polo · 周年 Tシャツ · ASN AS200261 commemorative · 創業の地 Amsterdam Tシャツ
  • ユーザー向け (Outside-in): Validator badge Tシャツ (個別 pubkey 焼込) · ERPC edge global map · "Green RPC" sticker · Skeet OSS contributor patch
  • 両方欲しがる: Solana co-brand Tシャツ · 「For Sustainable…」 manifesto Tシャツ · Shinobi top 3 commemorative cap
