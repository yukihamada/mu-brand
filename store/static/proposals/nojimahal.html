0. CONCEPT · BRAND IDENTITY

5 つのルールに 圧縮された 1 つのブランド

NOJIMAHAL は 5 つの規律 で動く。 多く語らない。 多く見せない。 1 つに絞る。 — その方が、 道場でも街でも目立つから。

  1. RULE 01 — 色は 2 色だけ
    Black + Volt。 第三色を入れた瞬間、 NOJIMAHAL ではなくなる。 グレーや白すら「使わない」と決めた領域。
  2. RULE 02 — Volt は accent のみ
    Volt は面積比 5% 以下に固定。 縫い目・ステッチ・刺繍・小ロゴ・スリーブ stripe — 「点で効かせる」 使い方のみ。
  3. RULE 03 — 角は丸めない
    UI / グラフィック・ボタン・タグ・ラベル — すべて border-radius: 0。 角が立っていることが規律。 ボックスのまま。
  4. RULE 04 — タイポは 1 種
    Helvetica Neue / Inter 900 condensed のみ。 装飾 typeface は使わない。 ロゴから footer まで同じ font で揃える。
  5. RULE 05 — 装飾は禁止
    グラデーション / 影 / 装飾フレーム は使わない。 1 着 = 1 〜 2 グラフィック (ロゴ + Volt stripe) のみ。 「足し算で説得しない」。
  6. — 由来 / NAMING —
    NOJI · MAHAL = Nojima + Mahal (Taj Mahal の "宮殿" を借りる)。 柔術家の宮殿を 1 着の中に再現する、 という宣言。

1. なぜ NOJIMAHAL — 一枚で 24h 横断する

柔術愛好家の 1 日は 道場 → カフェ → 仕事 → 夜の街 と切り替わる。 これまでは着替えが要った。 NOJIMAHAL は同じパレット (Black + Volt) と同じ素材語彙でこれを 1 着で繋ぐ。

オーバーサイズ mode-sport: 「スポーツの素材で mode を着る」 — ドロップショルダーの boxy fit を黒一色で統一する。 NOJIMAHAL は柔術というニッチに絞ることで、 唯一のニッチ no.1 を取りに行く。

野島繁昭プロデュース: BJJ practitioner として実用域の rashguard 仕様 (フラットロックシーム / 4-way stretch / 防臭) と、 mode collector としての都市着サイズ感 (oversize boxy / drop shoulder) を、 同じ人格の中で握る。

3 つの想定使途

  • 競技 + 都市 横断ワードローブ (野島 + 同志柔術家): 1 セット買い切りで道場 → 街を切れ目なく
  • 道場 / アカデミー uniform 兼販売: 黒道着の延長としてオレンジ帯クラス全員に
  • 大会 limited drop: 戦績入り 1-of-N (背中に着用者の戦績がプリント)
