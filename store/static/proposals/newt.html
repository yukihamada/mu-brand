1. なぜ MU からこの提案を

NEWT 様のプロダクトが体現するのは

「海外旅行を、 アプリで、 身軽に」

という体験。 重い旅行代理店のカウンターを、 手のひらの数タップに変えた。

MU の思想はこう重なります:

NEWT は『身軽に海外へ』を作る。 MU は『在庫を持たない = 身軽』で旅装を作る。

───

旅装の 3 軸 (空港 → 移動 → ビーチ)

① 着るもの (¥4,800-¥9,800) · 5 SKU — Tee / Long Sleeve / Hoodie / Crewneck。 機内で羽織れる軽量・しわになりにくい・パッカブル前提。 「身軽に、遠くへ」のワードマークと NW モノグラム。

② 運ぶもの (¥800-¥3,800) · 4 SKU — パッカブル Tote / Cap / Pouch (化粧ポーチ相当) / Luggage-tag 風 Sticker。 機内持ち込みに収まる軽量装備。

③ 旅のお守り (¥800-¥3,800) · 3 SKU — Mug (朝のホテルで一杯) / Phone Case (旅の相棒) / Sticker (スーツケースに貼る目印)。 QR / NFC で NEWT の旅程・行き先にリンクする「旅のお守り」構想。

───

無在庫 POD なので NEWT 様側に在庫・出荷義務は一切なし。 売上の 50% は MU §28 の標準寄付プールに回ります (NEWT 様限定の特定寄付先を勝手に紐づけることはしません — 寄付先は協議で決定可能)。 残り 25% が NEWT / 令和トラベル様、 25% が MU 運営。 月 1 回締めで Wise / 銀行振込。 配分は MU のデフォルト提案であり、 全て交渉可能です。

想定使途

  • NEWT アプリ利用者への旅装ノベルティ / プレミアム (空港→ビーチで動ける軽量装備)
  • 「身軽に、遠くへ」を体現する旅好きコミュニティのデイリーウェア
  • QR / NFC で NEWT の旅程・行き先にリンクする『旅のお守り』としての goods
  • 行き先 (destination) テーマの月相連動 drop (抽象テーマ・実ツアー名は不使用)
  • 海外配送に強い Printful EU 直送で旅先の現地受け取りも視野
  • 旅行(体験)を売る NEWT と、 旅装(物)を売る MU の非競合 cross-sell

3. 仕組み — 6 原則

01

旅行(体験)と旅装(物)は構造的に非競合

NEWT / 令和トラベル様が売るのは海外旅行という体験 (次世代 OTA・アプリ予約)。 MU が売るのは旅に持っていくアパレル・goods (物)。 顧客は同じ「旅好き」でも、 買うものが体験 vs 物で完全に別カテゴリ。 値下げ競争・市場食い合いは構造的に発生しない。 むしろ旅装が旅行体験への期待を高める cross-sell 関係。

02

『身軽』思想の一致 — 在庫を持たない = 旅も身軽

NEWT のコアバリューは『身軽に海外へ』。 MU のコアは『在庫を持たない無在庫 POD』。 両者とも「重いものを持たない」設計思想。 ライン名 「旅を、着る。 / 身軽に、遠くへ。」 はこの一致点を一行に圧縮したもの。 思想レベルで噛み合う collab。

03

実 POD 媒体で印刷できる商品のみ (誇大表示ゼロ)

全 12 SKU を Printful EU / Gelato JP で実際に印刷・発送できる媒体に限定。 Tee / Long Sleeve / Hoodie / Crewneck / Tote / Cap / Pouch / Mug / Phone Case / Sticker — すべて既存 POD ラインで実績のある商品。 「軽量」「パッカブル」は各媒体の実仕様の範囲で記載し、 出せない仕様で売ることは構造的に不可能。

04

QR / NFC は『構想』として正直に提示

QR / NFC で NEWT の旅程・行き先にリンクする『旅のお守り』 は魅力的な hook だが、 現時点では構想 (proposal) であり実装未確定。 NEWT 様の API / ディープリンク仕様の開示と合意が前提。 合意前は静的 QR (newt.net トップへの送客) に留め、 動的旅程リンクは Phase 2 として正直に分離。

05

売上 50% は MU 標準寄付プールへ (特定寄付先は捏造しない)

MU §28 の枠組みで売上 50% を MU 標準寄付プールに回す。 NEWT 様限定の特定チャリティ提携を勝手に作って謳うことはしない (誠実性原則)。 寄付先・配分は NEWT 様との協議で決定・変更可能。 残り 25% が NEWT / 令和トラベル様、 25% が MU 運営。 月締 Wise / 振込、 寄付実績は wearmu.com/newt/donations に公開予定。

06

ロゴ・行き先・役員名は一切捏造しない

本 LP のロゴ・配色・ビジュアルは MU が想定で作った仮モックであり、 NEWT 様の実ロゴ・実アセットは未使用。 ツアー名・行き先・役員名・社内 KPI など具体的な事業情報は一切記載していない。 「行き先テーマ」 の drop も実在ツアー名を使わず抽象的な月相連動 に留める。 許諾取得後に実ブランドアセットへ差し替える前提。
