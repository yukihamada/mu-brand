1. なぜ MU からこの提案を

CHAR 様の 15 年分の奄美の海 は、 一見アパレルや雑貨と相性が悪そうに見えます。 写真の解像度と階調が、 安易な T シャツ印刷で潰れるからです。

このコラボは 2 層構造で組みます。

① 写真をそのまま印刷する層 (昇華転写) — ビーチタオル / ブランケット / キャンバス / フォン ケース / マグ / 枕 / バンダナ / フラッグ / マウスパッド。 ポリエステル昇華転写は写真本来の発色をほぼ無損失で再現でき、 海・水中・夜空 のような繊細なグラデーションが活きる。 これらは CHAR 様自身の額装プリント市場とは異なる客層・価格帯 (¥980-¥18,000)。

② MU が monoline / silhouette に翻訳する層 (chest print アパレル) — T シャツ / キャップ / トート。 写真の構図だけを 1 本の線に落とし込み、 inner tag に photographer credit + QR で charfilm.com へ送客。 着る人が必ず写真家本人に辿り着く。

無在庫 POD なので CHAR 様側に在庫・出荷義務は一切なし。 売上 50% は奄美の自然保全 NPO に寄付、 25% が CHAR 様、 25% が MU 運営。 月 1 回 締めで Wise 振込。

3. 仕組み — 6 原則

01

既存ショップと競合しない

charfilm.thebase.in は触らない。 額装プリント (¥38,000-¥88,000) / 写真集 COLORS (¥8,800) は CHAR 様の核心事業として保護。 MU は 同じ写真の異なる媒体・異なる価格帯 (アパレル + 昇華 lifestyle + ¥980 magnet 等) のみを扱う。 値下げ競争には絶対にならない。

02

写真は無損失で印刷する

昇華転写ポリエステル (ビーチタオル / 枕 / バンダナ / フラッグ / マウスパッド) と キャンバス / 紙ポスター / フォンケース / マグ は写真本来の色域・階調をほぼ無損失で再現。 7 枚それぞれの最適媒体 を MU 側で選定済み (波 → タオル、 バレル → ケース、 鯨 → マグ、 星イルカ → ブランケット …)

03

アパレルは monoline に翻訳

T シャツ / キャップ / トート は写真をそのまま印刷せず、 MU が 1 本の線 (monoline) と シルエット に翻訳。 inner tag に「Inspired by CHAR / charfilm.com」 + QR を印字。 着る人が必ず CHAR 様 本人サイトに辿り着く。 写真 IP は CHAR 様、 monoline IP は両者使用可。

04

売上の 50% を奄美の自然保全 NPO へ

場所 = 奄美 という限定で MU §28 寄付 50% 枠を 奄美の自然保全 NPO (奄美野鳥の会 / 奄美水産研究所 / WWF 奄美 等から CHAR 様が選定) に redirect。 残り 25% CHAR 様 / 25% MU 運営。 寄付実績は wearmu.com/charfilm/donations に毎月公開。

05

全 SKU から charfilm.com へ送客

全 26 SKU の商品ページ /p/charfilm-* に 「写真家 CHAR (土屋 尚幸) の作品集を見る」 → charfilm.com リンク + 額装プリントへの誘導バナーを設置。 inner tag + 同梱カード にも QR 印字。 MU で買った人が 写真集 COLORS / 額装プリント の本流顧客に転換する導線を設計。

06

月相 drop · 在庫なし

Drop は月相連動 (朔 / 弦 / 望)。 在庫を持たないため drop 終了後は注文不可になり、 次月相 で再開。 限定感は自然に発生。 OND° クライアント案件 (NHK大河 / NALU 等) のリリース時期に合わせた特別 drop も柔軟に対応。
