CHAR 様が charfilm.com/about に書いた言葉:

「写真の力や可能性は、 人間の眼差しを変える力であり、 眼差しが変わることが意識を変え、 意識が変われば世界が変わり、 自分が生まれ変わる」

このコラボの一行はここから取ります。

1 枚の写真が眼差しを変える。 1 枚のキャンバスが部屋を変える。 1 冊の ZINE が日常を変える。

───

② アパレル AOP 軸 (¥7,800-¥12,800) — Athletic Tee (黄金の波 AOP) / Rashguard LS (透明バレル AOP) / Hoodie (光の柱 背面 AOP)。 すべて全面昇華転写ポリエステルまたは背面 AOP プルオーバーで、 DONT PANIC × CHAR FILM TEE (オーガニックコットン重厚 chest print) とは素材・印刷面積・用途が完全に異なる。 surf / SUP / ヨガ / ジムの濡れる・動く現場で使う機能服。

3. 仕組み — 7 原則

01

charfilm.thebase + DONT PANIC TEE 共に不可侵

charfilm.thebase.in (額装 ¥38-88K / 写真集 COLORS ¥8.8K / 1000部) と DONT PANIC × CHAR FILM TEE 5th installment (¥7,700 オーガニックコットン重厚 · YTS Store) は CHAR 様の既存事業として完全保護。 MU は異なる媒体・異なる用途・異なる国際配送 (Printful EU 直送 + 昇華転写 lifestyle + NEDI 50% 寄付) のみを担当。 値下げ競争・市場食い合いは構造的に発生しない。

02

写真は無損失で印刷する (lifestyle 軸)

昇華転写ポリエステル (ビーチタオル / 枕 / フォンケース / マグ) と キャンバス + 紙ポスター は写真本来の色域・階調をほぼ無損失で再現。 7 枚それぞれの最適媒体を MU 側で選定済み (ネリヤカナヤ夜明け → A2 キャンバス hero、 透明バレル → タオル + ケース + ラッシュガード、 鯨 → マグ、 イルカ星空 → ブランケット、 水中光柱 → フーディー背面、 黄金の波 → AOP Tee)。

03

アパレルは AOP / 機能服のみ (DONT PANIC と棲み分け)

MU の T シャツ / ラッシュガード / フーディーは全面昇華転写ポリ (Athletic Tee / Rashguard) または 背面全面 AOP (Hoodie)。 DONT PANIC × CHAR FILM TEE のオーガニックコットン重厚 chest printとは素材・印刷面積・用途が完全に異なる。 MU 軸 = surf / SUP / ヨガ / ジムで濡れて動く現場の機能服。

04

売上 50% は NEDI に固定

場所 = 奄美 という限定で MU §28 寄付 50% 枠を 一般社団法人 NEDI (amami-nedi.com · 代表 碇山勇生氏 · 奄美のサンゴ再生 / 海洋環境保全) に redirect。 CHAR 様が Patagonia「尊々加那志」で碇山氏自身を撮影した既存関係を活かす設計。 残り 25% CHAR 様 / 25% MU 運営。 寄付実績と NEDI 領収書は wearmu.com/charfilm/donations に毎月公開。

05

全 SKU から charfilm.com / thebase へ送客

全 12 SKU の商品ページ /p/charfilm-* に 「写真家 CHAR (土屋 尚幸) の作品集を見る」 → charfilm.com + 「額装プリント・写真集 COLORS を購入する」 → charfilm.thebase.in の二段リンク。 inner tag + 同梱カード にも QR 印字。 MU で買った人が CHAR 様の本流顧客に転換する導線を設計。

06

海から山への眼差し (Drop 04+ で拡張)

CHAR 様が 2026 年に書かれた「森林限界域千六百メートル以上には高い木は無く、 有機物と無機物が露骨に存在する世界はまるでどこかの惑星に降りたような気分だった」 (屋久島 blog) の通り、 現在 海から山へ眼差しが拡張中。 Drop 01-03 は奄美の海軸、 Drop 04+ は屋久島照葉樹林 / ヒマラヤ シリーズに拡張可能 (写真追加 + 同じ NEDI 寄付モデル流用)。 「海も山も、 同じ眼差し」 のシリーズ。

07

ネリヤカナヤ夜明け がアンカー

CHAR 様の既存 ¥77K アクリル + ¥88K 大額装 で繰り返し採用される ネリヤカナヤ夜明け を MU では A2 キャンバス 限定 50 部 signed + 番号 + COA (¥28,000) として hero に。 既存 ¥38K 小額装とサイズ・媒体・価格帯すべて異なる位置取りで、 「ネリヤカナヤを家に置きたいが ¥38K は今は出せない」 層への適切な入口。 50 部完売で NEDI に ¥700,000 寄付。
