3. 仕組み — 6 原則

01

charfilm.thebase + DONT PANIC TEE 共に不可侵

charfilm.thebase.in (額装 ¥38-88K / 写真集 COLORS ¥8.8K / 1000部) と DONT PANIC × CHAR FILM TEE 5th installment (¥7,700 オーガニックコットン重厚 · YTS Store) は CHAR 様の既存事業として完全保護。 MU は異なる媒体・異なる用途・異なる国際配送のみを担当。 値下げ競争・市場食い合いは構造的に発生しない。

02

印刷品質を解像度ベースで担保

全 12 SKU を 3600×2400 master file (CHAR 様提供想定) で印刷可能なサイズ・媒体に限定設計。 A2 Canvas で 154dpi (Printful 基準 150dpi 超え)、 A3 で 217dpi、 アパレル AOP で 100-180dpi (距離視認許容)、 ギフト系 (Mug / Phone Case / Postcard) は 300dpi 完全余裕。 印刷品質的に「これは間違いなく出せる」 範囲のみ。

03

アパレルは AOP / 機能服のみ (DONT PANIC と棲み分け)

MU の T シャツ / ラッシュガード / フーディーは全面昇華転写ポリ (Athletic Tee / Rashguard) または 背面大判 print (Hoodie 35×40cm)。 DONT PANIC × CHAR FILM TEE のオーガニックコットン重厚 chest printとは素材・印刷面積・用途が完全に異なる。 MU 軸 = surf / SUP / ヨガ / ジムで濡れて動く現場の機能服。

04

売上 50% は NEDI に固定

場所 = 奄美 という限定で MU §28 寄付 50% 枠を 一般社団法人 NEDI (amami-nedi.com · 代表 碇山勇生氏 · 奄美のサンゴ再生 / 海洋環境保全) に redirect。 CHAR 様が Patagonia「尊々加那志」で碇山氏自身を撮影した既存関係を活かす設計。 残り 25% CHAR 様 / 25% MU 運営。 寄付実績と NEDI 領収書は wearmu.com/charfilm/donations に毎月公開。

05

全 SKU から charfilm.com / thebase へ送客

全 12 SKU の商品ページ /p/charfilm-* に 「写真家 CHAR (土屋 尚幸) の作品集を見る」 → charfilm.com + 「額装プリント・写真集 COLORS を購入する」 → charfilm.thebase.in の二段リンク。 inner tag + 同梱カード にも QR 印字。 MU で買った人が CHAR 様の本流顧客に転換する導線を設計。

06

出来ないことは Phase 2 に分離 (誠実性原則)

「ネリヤカナヤ夜明け 限定 50 部 signed canvas」 「写真集 ZINE 海から山へ」 「150cm ビーチタオル」 等は本 LP の 12 SKU に含めない。 master file 解像度・サイン物流・印刷 partner 等が未確立なため。 これらは下記 「Phase 2 候補」 セクションで別記し、 各々の前提条件が揃った時点で個別合意で追加。 嘘の在庫表示・誇大な仕様で売る事は構造的に不可能。
