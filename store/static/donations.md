# §27 Donation Ledger — 弟子屈町 / 北海道

MU の Constitution §27 は、 落札額 (MA auction の captured 売上) の 50% 相当 (post-tax)
を、 北海道 川上郡 弟子屈町 に寄付することを約束しています。

このページは「実際に何円が、 いつ、 どこへ送金されたか」 の公開記録です。
銀行振込自体は人 (濱田優貴) が手作業で行い、 完了したら下のリストにエントリを追加します。

- **約束 (rule)**: `max(¥100,000, 50% × net-after-tax)` — fiscal year ごと
- **送金先 (default)**: 弟子屈町ふるさと納税 (返礼品なし) / 弥栄小学校・摩周小学校 等の特定指定先
- **記録粒度**: ふるさと納税 申込番号 or 銀行振込 ref を残す
- **見える化**: `/api/transparency` の `teshikaga_pledge.estimated_pledge_jpy` が今期の対象目安
- **コミット (Commit)**: このファイルへの追記は append-only。 過去エントリは削除しない (Constitution §13 の精神)
