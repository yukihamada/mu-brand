# JiuFlow 道場予約対応状況

オンライン予約に対応している道場は3つのみ:
- **Sweep** (福岡)
- **yawara**
- **Overlimit 札幌**

**Why:** 他の道場はJiuFlow予約システムに未連携。「全国の道場を予約」とは書かず、検索+掲載は全国対応、予約は一部道場のみ。

**How to apply:** LPやアプリで「道場を探して予約」と書く際、予約は上記3道場のみである点に注意。過大広告にならないよう「道場を探す」をメインCTAにする。
