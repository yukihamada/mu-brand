# SOLUNA / TAPKOP

## 事業構造（3社体制）
- **Enabler Inc.**（濱田優貴）— **Airbnbスーパーホスト**。物件運営・テック・ブランド・Webシステム（SOLUNAサイト, KAGI API, Pon電子契約）
  - 長年のスーパーホスト実績、複数物件で安定収益（LODGE年1,000万、熱海年1,800万 等）
  - **THE LODGE・WHITE HOUSE熱海は持分未販売（運営中だが共有持分はまだ売り出していない）**
  - **「完売」はTAPKOPのみ。LODGEとWHITE HOUSEを「完売」と書くのは誤り**
- **イーストサイド株式会社（East Side）** — TAPKOPの運営/開発担当
- **East Ventures（VC）** — **Enabler Inc.** に優先株5%で5,000万円出資済み（時価総額10億円評価）
- 役割分担: テック+運営(Enabler) × TAPKOP事業体(East Side) × 資本(East Ventures)
- **TAPKOPは濱田優貴と友人のTKさんの共同プロジェクト**（イーストサイド株式会社が事業体）

## 資金調達
- **Enabler Inc.**: East Ventures から優先株 **5%** で **5,000万円** 出資済み
  - → 時価総額ベースで 10億円評価
  - → "個人の趣味プロジェクト"ではなく、**正式な VC-backed startup**
  - 濱田本人も出資（「作る側」と「所有する側」を同一にする流儀）

## 物件書類Vault (2026-04-23 実装)

- DBテーブル: `property_documents`, `document_disclosures`
- 公開ページ: `/docs/<slug>` (書類の存在のみ公開、本文非公開)
- 保護DL: `GET /api/soluna/docs/:slug/:id/download` (nah_access=1 OR クーポン所有者 OR 開示済み)
- 管理者アップロード: `POST /api/soluna/docs/:slug/upload`
- 開示付与: `POST /api/soluna/docs/disclose` (`{doc_id, email}`)

  - ID1: village / 重要事項説明書（美留和 501番1ほか 106,899㎡）
