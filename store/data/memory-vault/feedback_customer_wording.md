顧客向けコピー（ブログ記事・配信メール・LP・アプリ UI・サポート文言）では「ユーザー」と書かず必ず「お客様」を使う。

**Why:** 2026-05-12、JiuFlow のアンケート反映ブログ/メールで「新規ユーザー向け」と書いていたら濱田氏から「ユーザーと書かずお客様って書いてね」と指摘あり。サービス提供者→顧客の関係性を尊重する書き方が JiuFlow / Enabler のトーン。

**How to apply:**
- 公開コピー（ブログ・メール・LP・FAQ・アプリ内UI文言）では「ユーザー」を「お客様」に置換
- 「新規ユーザー」→「初めてのお客様」、「ユーザーの声」→「お客様の声」、「ユーザーの皆様」→「お客様の皆様」
- 例外: 内部実装の変数名・コメント・技術文書（"user_ref" "user_id" など DB schema）はそのまま
- 例外: アンケート結果サマリの数値説明など客観記述では文脈に応じて「回答者」「ご利用者」など別語彙も可
- 対象プロダクト: JiuFlow / 同等の Enabler 系プロダクト全般（チャリン・パシャ・KAGI・サクッ等にも準じる）
