# 弟子屈町 企業版ふるさと納税（地方創生応援税制）

## 制度
- 法令: 地域再生法 + 租税特別措置法第42条の12の2
- 1回10万円以上の寄付が対象
- **損金算入(全額) + 税額控除** で最大約9割軽減（実質負担1割）
- 通常の地方公共団体寄付（法人税法37条3項1号、全額損金算入のみ）より約2倍お得

## 弟子屈町の認定事業（まち・ひと・しごと創生寄附活用事業）
- 摩周湖環境保全（水質調査、流域自治体連携）
- 観光農園運営・集出荷センター整備
- 川湯温泉街再整備（廃ホテル解体、宿泊施設誘致）
- 弟子屈町まち・ひと・しごと創生総合戦略に基づく事業全般
