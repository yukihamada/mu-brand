投資家の出資比率・評価額・出資額は、`MEMORY.md` インデックスの1行サマリではなく、必ず本体ファイル（[[soluna_tapkop]] 等）を読んで確認する。

**Why:** 2026-05-16、East Ventures の比率について `MEMORY.md` に「優先株10%」、本体 `soluna_tapkop.md` に「優先株5%、時価総額10億円評価、5,000万円」と矛盾があり、インデックスを引用して3回連続「10%株主」と発言。算術 (5,000万 ÷ 10億 = 5%) で検算すれば即座に矛盾検出できたが、しなかった。投資家持分率は法人設立の出資契約条項、事業切り出し時の希薄化判断、資金調達ストーリーに直結する数字で、間違えると意思決定を歪める。

**How to apply:**
- VC・株主・キャップテーブルの数字を発言する前に、本体ファイルを Read で開いて確認
- 「出資額 ÷ 評価額 = 比率」が成り立つか必ず暗算検算
- インデックスと本体が矛盾する場合は **本体が正**、即座にインデックスを修正
- 「○%株主」と書くときは出資額と評価額のセットで提示 (例: 5%/¥5,000万/時価10億)
