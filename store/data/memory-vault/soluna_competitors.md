# SOLUNA 競合マップ（2026-05-06調査）

## 想定ポジション
SOLUNA: 弟子屈・杉CLT+籾殻+竹SIPs・坪200-300万円帯（10-25㎡で本体600-2,250万円ゾーン）

**Why:** 性能だけでは坪200-300万円は説明不能。土屋ホームPremierがUA0.19を坪80万円台で売っている市場での価格設定。

**How to apply:** ローカル土地パッケージ・建築家コラボ・宿泊運用ファイナンスのいずれかを必ず積む。

2. **VUILD NESTING**（建築家×CNC・UA0.44・¥1,000-2,000万/12-24坪）— SOLUNAと完全に思想競合
3. **タイニーホーム北海道**（寒冷地専業・¥520万〜・トリプルサッシ）— 弟子屈現地での直接競合
4. **Koto (UK)**（£35K-660K・「日本×北欧」を先取り・Niwa/Pari等の和語ネーミング）— 海外で同コンセプト

1. 数値の徹底公開（UA/C値/LCA）
2. 価格の積み上げ式（住箱方式: 本体→標準→フル）
